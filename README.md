# TechSolutions - Sitio Web HTML5

Sitio web informativo para TechSolutions, empresa ficticia de desarrollo web.

## Estructura del Proyecto

- `index.html` - Página principal
- `servicios.html` - Página de servicios con tabla de precios
- `contacto.html` - Página de contacto con formulario completo
- `portafolio.html` - Página de portafolio con proyectos

## Tecnologías

- HTML5 puro (sin CSS ni JavaScript)
- Elementos semánticos
- Formularios con validación HTML5 nativa
- Tablas semánticas
- Accesibilidad (ARIA)
